## 🔀 Versionamento

- **`main`** – código estável e concluído  
- **`develop`** – integração antes de ir para `main`  
- **`feature/<assunto>-<exercicio>`** – branches para cada exercício 

### Padrão de commits
```
<type>(<escopo>): <descrição>
```
Exemplo: `feat(operadores): adiciona exercício 6 - maior de três números`
